# refuel-optimizer

## IE532 | Network Optimization | UIUC

This project is geared towards finding optimal refuel locations for a vehcle on a long tour where the fuel costs vary across cities and states.
Check report for more details.

## Main Dependencies
1. Networkx
2. osmnx
3. gurobipy

## To Run:
1. Install the required dependendencies
2. Open refuel.ipynb to specify area of interest
3. The code will find least cost fuel path between extreme top left to bottm right locations
